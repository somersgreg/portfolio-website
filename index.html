Sprint Challenge: User Interface and Git - Multi-Page Website

Sprint Challenge: User Interface and Git - Multi-Page Website This challenge allows you to practice the concepts and techniques learned over the past week and apply them in a concrete project. This Sprint explored User Interface and Git. During this Sprint, you studied Semantic HTML, CSS Fundamentals, CSS Flexbox Module, and Git. In your challenge this week, you will demonstrate proficiency by creating a multi page website that has some missing HTML elements as well as CSS specificity problems that need to be solved. You will also create an additional web page that will be linked to from a navigation you will build.

User Interface Project III: Great Idea Web Page With Flexbox

User Interface Project III: Great Idea Web Page With Flexbox Now that you have experienced building a layout using inline-block. Lets revisit the same HTML structure and refactor our first page with flex box module. If you didn't finish yesterday's assignment, just start from scratch or use the solution code provided to you. Once you have finished the home page using flex box module, go build the services page based on the design file provided to you. This time you will need to structure your own HTML in addition to using flex box module. You have been provided all the necessary images in the img folder.

Responsive Web Design I: Adaptive Website

Responsive Web Design I: Adaptive Website It's time to expand our CSS skills into multiple devices. You will be creating an adaptive version of the "Great Idea" design. You will be working through design mock ups for desktop, tablet, and mobile devices located in the design-files folder.

Preprocessing I: Digital Resume

Preprocessing I: Digital Resume For this project you will be building a digital resume from scratch. You will be required to use specific preprocessing skills to accomplish your tasks.

Build A Portfolio Website

Build A Portfolio Website With all of the projects you will be working on here at Lambda School, you will need somewhere to house them! There is no better opportunity to learn and show off your skills than a beautiful portfolio website. You have the ability to write HTML, CSS, and responsive media queries. You also know how to identify and write responsive units. It's time to put that knowledge into action by reading someone else's code and adding your own flavor. You will be building a portfolio website from a template found on https://html5up.net/.
